Project Overview

This project focuses on building and evaluating machine learning models to predict credit default risk. The system compares traditional statistical models with ensemble learning approaches and studies how credit behavior differs across economic contexts.

Datasets Used

  • Indian Credit Dataset: Bureau-style anonymized data representing borrower repayment behavior.
  • US LendingClub Dataset: Public peer-to-peer lending data with loan and borrower attributes.

Models Implemented

  • Logistic Regression (with threshold calibration)
  • Random Forest Classifier

Evaluation Metrics

  • ROC-AUC Score
  • KS Statistic
  • Recall (Default Class)
  • Precision (Default Class)

Key Insights

  • Indian credit data shows strong linear separability, making Logistic Regression highly effective.
  • US credit data exhibits non-linear patterns and policy sensitivity, benefiting from ensemble models.
  • Model performance is influenced more by economic context than algorithm choice alone.

Future Enhancements

  • SHAP and LIME for model explainability
  • NLP-based borrower text analysis
  • End-to-end automated loan decision system
