Project Overview

This project simulates a real-world credit risk evaluation system used by financial institutions to assess borrower default probability. By comparing linear and ensemble models across different economies, the system highlights how credit behavior is shaped by regional financial structures and policy environments.

Evaluation Metrics

  • ROC-AUC Score
  • KS Statistic
  • Recall (Default Class)
  • Precision (Default Class)

Key Insights

  • Indian credit data shows strong linear separability, making Logistic Regression highly effective.
  • US credit data exhibits non-linear patterns and policy sensitivity, benefiting from ensemble models.
  • Model performance is influenced more by economic context than algorithm choice alone.

Future Enhancements

  • SHAP and LIME for model explainability
  • NLP-based borrower text analysis
  • End-to-end automated loan decision system
